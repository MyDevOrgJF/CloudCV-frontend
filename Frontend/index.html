Projects

Improved Spotify Shuffle

I built a React-based web app to improve Spotify shuffle playback. I was annoyed with how the Spotify shuffle playback was not very random at the time, it would repeat recently played songs on my large playlists. I improved it by saving locally data of played songs with a date and time played. Using a function I made in my app, I can auto skip songs last played in a selectable date range. To interact with Spotify, and retrieve their data I needed I used Spotify’s API and player SDK. It runs in web browsers. I styled it with CSS and made the view adaptive for mobile and PC resolutions. It has a server layer that runs on Node to make the API calls. Deployed on Fly.io.

www.ShufflePlus.app | Github repo

Azure dynamic website CV

I built an Azure-based project after achieving the Microsoft AZ-900 certification, to demonstrate practical cloud skills and applied knowledge. It is a static website version of my CV hosted on Azure. On the frontend it is html, CSS and JS. A JavaScript page visitor counter stores and retrieves data from an Azure Cosmos DB using an Azure Functions API written in C# with unit tests. Azure Frontdoor was used to configure my custom domain name. Backend and frontend code are on separate GitHub repositories with GitHub Action workflows setup for CI/CD. So, whenever I make a change to either repo it automatically deploys to Azure after running tests.
